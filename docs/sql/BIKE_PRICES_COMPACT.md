| Байк | Почему подходит |
|------|----------------|
| **Falcon Pro / Falcon GT / Falcon Lynx** | Класс М — права не нужны или подходят B/A1. Вес 55-69 кг, низкая посадка. Электро — нет переключения передач |
| **Ducati Panigale S Electro** | Класс М — права B или M1. Тяжёлый (150 кг), но едет плавно, электромотоцикл |

## 🔧 Все ценовые ключи в specs JSON

- `dailyPrice` — аренда сутки (будни)
- `rent_weekend` — аренда сутки (выходной)
- `price_per_hour` — 1 час
- `price_per_3h` — 3 часа
- `price_per_6h` — 6 часов
- `price_per_12h` — 12 часов
- `rent_2_4d` — 2-4 дня
- `rent_5_10d` — 5-10 дней
- `rent_11_30d` — 11-30 дней
- `sale_price` — цена продажи
- `price_rub` — рекомендованная розничная цена
- `access_tier` — уровень доступа (`entry` / `pro` / `standard`)

### Поля характеристик

- `power_kw` / `power_hp` — мощность
- `weight_kg` — снаряжённая масса
- `seat_height_mm` — высота по седлу
- `top_speed_kmh` — макс. скорость
- `engine_cc` / `bike_engine_cc` — объём ДВС
- `range_km` — запас хода (для электро)
- `license_class` — категория прав
- `bike_subtype` — тип мотоцикла
- `access_tier` — грейд: `entry` / `mid` / `pro` / `standard`
