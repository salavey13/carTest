## Key Design Decisions

### Root Landing Page (app/page.tsx)
- **Static Server Component** - No "use client", no DB fetch, no Telegram init
- **Hardcoded 6 curated bikes** representing all categories:
  - Electric Entry: Falcon Pro 2025 (10 000₽/day)
  - Electric Premium: Y-VOLT Surge V (12 000₽/day)
  - Electric Sport: Ducati Panigale S Electro (10 000₽/day)
  - ICE Liter: Suzuki GSX-S1000F (14 000₽/day)
  - ICE Entry: Nibbler 300 4V (6 000₽/day)
  - Mini/Commute: Sotion EM01 (Sale 100 000₽)
- **Preserve Max's good parts**: VIP_BIKE_THEMES, SOCIAL_LINKS, CONTACT_INFO, FAQ_ITEMS, HOW_IT_WORKS_STEPS, MagneticButton
- **Fix pricing**: "от 1 500 ₽" → "от 6 000 ₽" (real lowest price)
- **Section order**: Hero → **Social Links** → Barrier Cards → Bike Showcase → How It Works → Pricing → FAQ → Contacts → Footer
- **Aesthetic**: Gold (#FFD700) on black (#0A0A0A)

### Loading Component (app/loading.tsx)
- Replace bicycle SVG with S1000RR GIF from Supabase
- CSS filter: `invert(1) sepia(1) saturate(2) hue-rotate(5deg)` (black→white, then gold tone)
- URL: `https://inmctohsodgdohamhzag.supabase.co/storage/v1/object/public/carpix/Loader-S1000RR-8cb0319b-acf7-4ed9-bfd2-97b4b3e2c6fc.gif`

### Electro-Enduro Subpage Filtering
- **Filter by `specs.type === "Electric"`** (not name matching)
- This excludes ALL ICE bikes (Nibbler, Motoland, Suzuki, Kawasaki)

### Main Franchize Page Categorization
- **Three groups with specific titles**:
  1. Electric bikes → **No title** (just show)
  2. ICE for Sale (Nibbler, Motoland) → **No title** (just show)
  3. ICE Rent-Only (Suzuki, Kawasaki) → **"Байки партнёров"**
- **Data-driven using**: `specs.type === "Electric"`, `specs.type === "ICE" && specs.sale === true`
- **No literally "ICE" text** - Russian audience won't understand

### Spec Display (CatalogClient + Item Modal)
- **Russian labels from `spec_labels` field** in rawSpecs
- **Consistent 3 priority specs**:
  - Electrics: Power (кВт) + Speed (км/ч) + Range (км)
  - ICE: Power (л.с.) + Speed (км/ч) + Engine (см³)
- **Fallback fields**: `power_hp || bike_power_hp`, `engine_cc || bike_engine_cc`

### Social Banner Design
- **Gold-on-black aesthetic** - no rainbow gradients
- **All icons converted to gold** - unified cyber-gold vibe
- **Placement**: Immediately after hero ("BAM! SOCIAL LINKS")
- **CSS**: `border: 2px solid #FFD700`, hover: `box-shadow: 0 0 30px rgba(255, 215, 0, 0.4); transform: scale(1.05)`

## Technical Constraints
- **Server Components only** - No event handlers (onMouseEnter, onMouseLeave)
- **CSS-only hover effects** using `group-hover` pattern
- **No styled-jsx** - Use Tailwind classes or inline styles
- **Instant load** - No blocking Telegram/Supabase calls
- **VipBikeRental page**: Added `export const dynamic = 'force-dynamic'` to prevent build timeout
