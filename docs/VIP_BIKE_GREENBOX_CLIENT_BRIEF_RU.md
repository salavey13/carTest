# VIP-BIKE + GREENBOX — коротко и по-человечески

Дата: 20 марта 2026  
Для кого: партнёры, команда, новые участники.

## Что уже есть прямо сейчас

### VIP-BIKE
- Уже работает публичная витрина по ссылке вида `/franchize/[slug]`.
- Можно смотреть каталог, открывать карточки, складывать в корзину и оформлять заказ.
- Есть страница настройки стиля и контента (`/franchize/create`) для команды без кода.

### GREENBOX
- Уже есть отдельная зона `/greenbox` с понятной демо-логикой.
- База и очереди полива в БД уже подготовлены.
- Демонстрация понятна даже людям без технического опыта.

## Почему это хорошо для клиента

- Продукт уже живой, не просто презентация.
- Есть понятный план на ближайшие шаги.
- Команда может работать параллельно и не мешать друг другу.
- VIP-BIKE даёт путь к выручке, Greenbox — удержание и вау-эффект.
