Почему это круто

⚡

Мгновенные ответы

Curl к Supabase REST API — ответ за ~2 секунды. Без открытия браузера, без Telegram WebApp, без ожидания загрузки React.

🔒

PII маскирование

Телефоны +7XXXXXXXX42, паспорта XXXX…1234, email i…@x.ru. Сервисный ключ Supabase никогда не попадает в вывод.

🧠

Anti-hallucination

Никаких выдуманных ID, цен, статусов. Все данные — из живой базы. Если чего-то нет, навык так и скажет.

🕐

Moscow TZ-safe

Все даты — Europe/Moscow. Между 00:00 и 03:00 не будет «вчерашнего» дня. Сравнения дат корректны.

🤖

9 Boss-команд

Утренняя сводка, вечерний отчёт, watchdog-алерты (просрочки, QR, застрявшие лиды) — автоматически в Telegram.

🔄

Self-enhancement

Ассистент может активировать factory-навыки (Avito, видео-генерация, .docx), создавать новые, вести свои KPI.
