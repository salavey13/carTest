SET metadata = jsonb_set(

  '{franchize}',

Every achievement is etched into git history. Not a database that can be wiped. Not a server that can go down. A commit. As permanent as code gets.

Git commits = eternal record (as long as GitHub is up). The blockchain of real work. Every door opened, every pill taken, every score earned — all in the commit log.
